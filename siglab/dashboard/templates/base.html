SigLab Help

Data Model

Run
A single research session. Contains multiple experiments evaluated in sequence.
Experiment
A strategy hypothesis tested against historical data. Has a spec hash, family, and evaluation metrics.
Track
A category of strategies: Directional Perps (trend-following) or Systematic Carry (yield-based).
Family
A specific strategy template (e.g., momentum_carry, pair_trade). Each family has its own feature set and parameters.
Spec Hash
Unique identifier for an experiment's configuration. Same spec hash = same strategy parameters.

Metrics Glossary

Overall Score
Primary selector metric. Combines Sharpe, return, and stability into a single score.
Median Sharpe Ratio
Risk-adjusted return. Higher is better. >1.0 is good, >2.0 is excellent.
Median CAGR
Compound annual growth rate. Annualized return across selector windows.
Median Return
Total return across selector windows. Not annualized.
Pre-Audit Return
Return measured before the audit boundary. Used for in-sample evaluation.
Median Calmar Ratio
Return / max drawdown. Higher is better. Measures how much return per unit of drawdown risk.
Validation Return
Out-of-sample return on validation slices. More trustworthy than in-sample metrics.
Validation Sharpe
Out-of-sample Sharpe on validation slices.
Audit Return
Final untouched out-of-sample return. The most trustworthy metric — data the strategy has never seen.
Audit Sharpe
Final untouched out-of-sample Sharpe.

Common Workflows

Create a new run
python3 -m siglab run --track trend_signals --iterations 1
View the ops board
Navigate to /ops or click "Ops" in the navbar.
Export data
Use the Export CSV/JSON buttons on the dashboard, or /api/export/runs?format=csv.
Search for a run or experiment
Press Cmd+K or / to open the command palette. Type to search.
Keyboard shortcuts
R refresh · / search · 1-7 ops panels · Esc back
