# Workspace Guide

Use the workspace as a file system, not as one giant prompt.

- `current/` holds the cheap always-loaded state.
- `manifests/` holds family constraints and allowed surfaces.
- `cookbooks/` holds named pattern groups.
- `cards/experiments/` holds canonical pre-audit spec summaries.
- `cards/reflections/` holds compact lesson cards.
- `cards/probes/` holds cached evidence outputs.
- `indexes/*.jsonl` are the search surface.

Only open deeper files when the default context is insufficient.
